# Perception Learning Platform

Modernization of an educational psychology learning application originally developed for Windows XP into a modern multilingual web learning platform.
 
Developed as part of an Advanced Software Engineering Practical Course at Philipps University Marburg.

Official project title:

"Von Windows XP ins Web: Portierung einer Lernanwendung zu Wahrnehmungseffekten"

---

## Project Overview

This project focused on migrating a legacy educational application from a Windows XP environment to a modern browser-based platform.

The original software was used in psychology education to demonstrate and explain various visual perception effects through interactive learning modules and simulations.

The goal was to preserve the educational content while redesigning the application using modern web technologies and making it accessible across different devices and operating systems. 

The resulting platform supports multilingual content, reusable web components, responsive layouts, and interactive perception experiments.

---

## Project Background

The development was organized across multiple student teams.

The first team established the initial web architecture and project foundation.

I worked in a two-person development team responsible for extending the platform, improving the architecture, implementing perception modules, and migrating learning content from the original application.
